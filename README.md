## Entorno virtual

1. Crear un entorno virtual:
```python
    python3 -m venv nombre_del_entorno
```

2.  Activar el entorno virtual en macOS y Linux:
```python
    source nombre_del_entorno/bin/activate
```

3. Desactivar el entorno virtual:
```python
    deactivate
```
